Module: routes/index

This file defines the routes for the Budget Claims System application. It includes routes for home, login, register, dashboard, help, user management, reports, logout, audit logs, and import data. The routes are protected by authentication and role-checking middleware where necessary. Sensitive data in request bodies is filtered out before logging.
